## 1. 说明

所有操作都在root权限下执行

## 2. 命令

### 2.1 下载

```bash
make download
```

### 2.2 清理下载目录

```bash
make clean
```

### 2.3 安装

```bash
make install
```

### 2.4 卸载

```bash
make uninstall
```

### 2.5 设置常用用户

```bash
make user USER=qingbing
```
