## 1. 说明

docker-compose 脚步都在限下执行, 执行本命令的前提是需要自行安装好 docker

## 2. 命令

### 2.1 下载

```bash
make download
```

### 2.2 清理下载目录

```bash
make clean
```

### 2.3 安装

```bash
make install
```

### 2.4 卸载

```bash
make uninstall
```
